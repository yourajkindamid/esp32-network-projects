# ESP-32 NETWORKING PROJECTS
### Bored over the summers and trying to make ESP-32 + networking projects.
- A web server which runs on an ESP-32. Used to light up LEDs. Will require ESP-32 Dev Module V1, breadboard, LEDs and resistors.
- A three wheeled wifi conrtolled RC car. The car itself requires a chassis, esp32, batteries, switch, and motor driver (optional: LEDs). Just upload the code onto the esp. The main point of using the esp is to use it as an access point with its own ssid (can add a password as well). Connect to the esp access point (i.e wifi) through a phone or laptop.
